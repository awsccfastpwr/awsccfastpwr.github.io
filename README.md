# AWS Cloud Club FAST Peshawar

Official website for AWS Cloud Club at FAST National University Peshawar.

**Live:** [awsccfastpwr.github.io](https://awsccfastpwr.github.io)

## Tech Stack

- **Next.js 15** (App Router, static export)
- **Tailwind CSS 4**
- **Framer Motion**
- **TypeScript**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Automatically deployed to **GitHub Pages** on every push to `main` via GitHub Actions. No external hosting required.
